# Capstone Theme
Story API Logger

## Project Concept
The main theme of my capstone project is a story API logger. 
The main reason why I chose that theme is because I began to write Webtoon/Manga stories since
2023 and I really love creating these types of stories/oneshots and a API logger will help me see the status of 
each of my projects and will help keep track on which story is in development, is ready for an artist to 
draw or is ready to be published. Another reason why I chose that theme for my capstone is because it 
will help me with creating my personal portfoilo and will have real-world application for me.
